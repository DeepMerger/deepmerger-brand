## The Mark

Three strokes — heavy, medium, light — converging to a single teal anchor point. The strokes represent every data source: weather, satellite, OSINT, terrain, units. The weight difference encodes intelligence hierarchy. The teal square is the moment of fusion: the Common Operating Picture.

**No gradients. No circles. Pure geometry. Scales from 16px favicon to 10-foot banner.**

## Typography

- **Display:** Space Grotesk 600 — wordmark, headings
- **Data/Mono:** DM Mono 400 — coordinates, timestamps, codes
- **Body:** Space Grotesk 400 — UI text

---

## Usage Rules

- Minimum clear space: mark height × 0.5 on all sides
- Minimum size: 16px (mark only) / 80px (with wordmark)
- Never stretch, rotate, recolour, or add effects to the mark
- Never use on backgrounds that reduce contrast below 4.5:1
