## Brand Concept

DeepMerger's mark represents the convergence of multiple intelligence streams into a unified operational picture. Three strokes — heavy, medium, and light — symbolize diverse data sources (weather, satellite, OSINT, terrain, units) merging at a single teal anchor point. The stroke weights encode intelligence hierarchy: primary data, secondary signals, and tertiary context. The teal square signifies the moment of fusion — total clarity from disparate inputs.

**Design Principles:** No gradients. No circles. Pure geometry. Infinite scalability.

## Color Palette

|------------|----------|-------|
| Deep Navy | `#0D1B2A` | Primary backgrounds, headings, dark text |
| Convergence | `#00C9A7` | Brand accent, fusion point, highlights |
| Command | `#1A3A4A` | Secondary surfaces, borders |
| Signal | `#E8EDF2` | Light text on dark backgrounds |
| Alert | `#FF4545` | Critical notifications, warnings |

---

## Typography

- **Display Font:** Space Grotesk 600 — Used for logos, headings, and prominent text. Letter-spacing: 0.18em.
- **Monospace Font:** DM Mono 400 — For data displays, coordinates, timestamps, and code.
- **Body Font:** Space Grotesk 400 — General UI text and descriptions.

---

## Usage Guidelines

- **Clear Space:** Maintain a minimum clear space of 0.5 × mark height on all sides.
- **Scalability:** The mark is designed to scale infinitely without loss of quality.
- **Color Variations:** Use appropriate variants for different backgrounds to ensure readability.
- **Do Not:** Alter the mark's proportions, add effects, or use unauthorized colors.
